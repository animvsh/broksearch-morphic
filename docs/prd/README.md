# Brok Product Requirements

Last updated: 2026-05-19

This directory contains implementation-grounded PRDs for Brok's current product surfaces.

- `brokmail-prd.md`: email client, Gmail/Calendar, Pi assistant, approvals.
- `brokcode-prd.md`: cloud builder, TUI, preview/deploy, GitHub, `brok-code` APIs.
- `integrations-prd.md`: Composio connector system and popup flows.
- `discover-library-spaces-prd.md`: saved research, knowledge library, spaces, task ledger.
- `platform-api-usage-billing-prd.md`: API keys, docs, playground, usage, billing, admin.
- `tools-and-subproducts-prd.md`: tools index, AI Humanizer, feature request widget.
- `insforge-brokcode-backend-prd.md`: InsForge as BrokCode's backend provider.

Linear-ready issue payloads live in `../linear-import/brok-prds-linear-import.json`.
